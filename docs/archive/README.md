# Archived planning docs

These are **historical** planning/roadmap documents from the early design of
NetVerse AI (formerly "InfraPulse"). They are **frozen** — kept for context on original intent, not
maintained and not accurate to what shipped (e.g. they mention Postgres/
TimescaleDB/Redis/Celery/WebSockets and "stub" collectors; the delivered system
uses SQLite + resilient background pollers with real pyVmomi/Netmiko/Genie
collectors).

For current information see:
- [`../../README.md`](../../README.md) — what the portal is and how to run it.
- [`../DEVELOPMENT.md`](../DEVELOPMENT.md) — the living feature-development playbook.

Do not edit these files. Track remaining/new work as GitHub issues.
